Overview

A fully integrated smart hospital automation system built on an Arduino microcontroller platform, incorporating 12 independent subsystems operating in real-time under a unified Bluetooth-controlled architecture. The system addresses environmental monitoring, access control, safety, lighting automation, energy management, and patient biosignal acquisition — all coordinated through a single mobile application.

💡 Relevance to Smart Lighting Infrastructure: The lighting control subsystems (PIR-based occupancy detection, Ultrasonic person-counting with automated illumination, and LDR-driven solar tracking) are directly applicable to intelligent urban lighting design — the core domain of companies like Schréder.
Integrated Systems
SYSTEM 04 / 05
Automated Lighting Control
Ultrasonic HC-SR04 × 2 + PIR Sensor
Bidirectional entry/exit counter triggers lighting automatically. PIR detects occupancy in corridors — lights activate only when presence is confirmed.
⭐ Smart Lighting Relevance
SYSTEM 07
Green Power System
Solar Panel + Dual LDR + Servo Motor
Dual LDR-based solar tracker with servo actuation — automatically orients panel toward maximum light intensity. Drives energy-efficient LED lighting loads.
⭐ Energy Efficiency Relevance
SYSTEM 01
Air Conditioning Control
DHT11 — Temp & Humidity Sensor
Environmental monitoring with relay-controlled fan actuation. Triggers at threshold temperature (>30°C) using analog threshold logic.
SYSTEM 02
Automated Door + Garage Lighting
Active IR Sensor + Servo Motor + Relay
IR proximity detects approaching vehicle → door opens automatically → lighting activates for a timed duration sufficient for parking → lights auto-off once driver exits. Eliminates idle lighting waste entirely.
⭐ Presence-Based Lighting
SYSTEM 03
Fire Alarm + Suppression
Flame Sensor + Buzzer + Water Pump
IR flame detection activates visual (LED), auditory (buzzer), and mechanical (water pump) responses simultaneously. 60° detection angle.
SYSTEM 06
Secure Door Lock
4×4 Keypad + Servo + LCD
Password-protected access control system with real-time LCD feedback, auditory wrong-entry alert, and reprogrammable password via keypad.
SYSTEM 07
Water Level Detection
Water Level Sensor — Hemodialysis Tanks
Resistive analog sensor monitors tank fill level. Automated pump activates when level drops below threshold — applied to hemodialysis water supply.
SYSTEM 08
Suction Air / Smoke System
MQ-2 Gas & Smoke Sensor
Metal oxide semiconductor sensor detects combustible gases (200–10,000 ppm). Triggers ventilation fan, LED indicator, and buzzer on detection.
SYSTEM 09
Sound-Triggered Control
Sound Sensor — LM393 Op-Amp
Adjustable sensitivity sound detection via LM393 comparator. Controls lighting state via acoustic trigger — touchless room control interface.
SYSTEM 10
Bluetooth Mobile Control
HC-06 Bluetooth Module
Unified wireless interface enabling real-time monitoring and control of all subsystems from a single mobile application via serial Bluetooth communication.
SYSTEM 12
Biosignal Acquisition
AD8232 ECG Sensor Module
Low-power (170µA) cardiac signal acquisition with instrumentation amplifier front-end. CMRR of 80dB ensures signal integrity in electrically noisy hospital environments.

Key Outcomes
  • ◆ Designed and implemented 12 independently functional subsystems operating concurrently on a single Arduino UNO, demonstrating efficient resource management and interrupt-driven logic.
  • ◆ Achieved full-system wireless control via Bluetooth mobile application, enabling unified monitoring and actuation of all subsystems from a single interface — applicable to IoT-based smart infrastructure management.
  • ◆ Implemented occupancy-based lighting automation using PIR and Ultrasonic sensors — a technique directly applicable to energy-efficient smart street and building lighting systems.
  • ◆ Developed a dual-axis solar tracking system using differential LDR readings, optimizing energy harvesting efficiency — aligned with sustainable smart city infrastructure goals.
  • ◆ Applied full product development lifecycle: requirements definition, component selection and BOM costing, circuit design, prototype assembly, software development, and functional testing.
